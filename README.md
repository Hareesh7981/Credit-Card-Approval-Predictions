# Credit-Card-Approval-Predictions

## Project Overview

This project predicts whether a customer's credit card application should be approved or rejected using Machine Learning. The model is trained on historical application and credit records to assist financial institutions in making faster and more accurate decisions.

---

## Features

- Data Cleaning and Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Random Forest Classifier
- Model Evaluation
- Confusion Matrix
- ROC Curve
- Feature Importance Visualization
- Trained Model Export (.pkl)
